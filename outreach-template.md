# Template Outreach — Clubs Tennis

**Objet :** Offre spéciale : Analyse vidéo gratuite pour vos élèves

**Corps de l'email :**

```
Bonjour [Nom du club / Responsable],

Je m'appelle [Sami] et je développe Tennis Breakdown, une plateforme d'analyse vidéo pour les joueurs de tennis amateurs.

Nous aidons les joueurs à progresser grâce à des analyses vidéo détaillées (technique, tactique, statistiques). Notre technologie identifie automatiquement les points forts et les axes d'amélioration.

**Offre spéciale pour votre club :**
- 3 analyses vidéo GRATUITES pour tester le service
- Tarif préférentiel pour vos membres : -20%
- Démo personnalisée possible sur place

Vos élèves pourraient filmer leurs matchs/entraînements et recevoir des recommandations précises pour progresser plus vite.

Intéressé(e) par un partenariat ? Je peux vous envoyer plus de détails ou organiser une courte présentation (15 min).

Cordialement,
Sami
Tennis Breakdown
[ton téléphone]
[ton site]
```

**Variantes possibles :**
- Sujet : "Partenariat club de tennis - analyses vidéo"
- Sujet : "Améliorez les performances de vos joueurs avec l'analyse vidéo"
- Ajouter : "Nous avons déjà aidé [X] joueurs à progresser"
