# Campagnes Facebook/Instagram Ads - Plan Détaillé

## Campagne 1: Découverte (Awareness)

### Audience
- **Intérêts**: Tennis, Roland Garros, ATP Tour, WTA, Novak Djokovic, Rafael Nadal
- **Comportements**: Achat équipement sport, participants à des cours
- **Démographie**: 18-55 ans, France, Belgique, Suisse, Canada (francophone)
- **Exclusions**: A déjà visité site, convertis

### Ad Sets (3 variations)

#### Ad Set A: Problème/Solution
- **Ciblage**: Joueurs compétitifs (intérêts tournois, ATP)
- **Message**: "Stagné dans votre classement? Découvrez pourquoi."
- **CTA**: Télécharger guide gratuit

#### Ad Set B: Résultats
- **Ciblage**: Joueurs passionnés (intérêts équipement, coaching)
- **Message**: "Améliorez votre technique de 30% en 2 semaines."
- **CTA**: Voir exemples d'analyses

#### Ad Set C: Curiosity
- **Ciblage**: Large audience tennis
- **Message**: "Les 5 erreurs que vous faites sans le savoir."
- **CTA**: Découvrir erreurs

### Ads Creative
#### Format 1: Vidéo avant/après
- **Durée**: 30 secondes
- **Content**: Transformation visuelle (floutée pour confidentialité)
- **Text overlay**: "Avant analyse" / "Après corrections"
- **Music**: Énergique mais non distrayante

#### Format 2: Carousel processus
- **Images**: 4-5 cards montrant le processus
  1. Upload vidéo
  2. Analyse experte
  3. Commentaires détaillés
  4. Résultats amélioration
- **Copy**: "Comment ça marche en 4 étapes simples"

#### Format 3: Testimonial short
- **Format**: Vidéo 15 secondes
- **Content**: Client satisfait (flouté) + citation texte
- **Social proof**: "Note 4.9/5 sur 50+ analyses"

### Budget & Schedule
- **Budget quotidien**: €10 par ad set (€30 total)
- **Durée test**: 7 jours
- **KPI**: CPC < €1.50, CTR > 2%
- **Optimisation**: Après 3 jours, pause sets non performants

## Campagne 2: Conversion (Consideration)

### Audience
- **Custom Audience**: Visiteurs site (30 jours)
- **Lookalike**: 1% similarité aux convertis (quand disponibles)
- **Engagement**: Interagi avec contenu précédent
- **Exclusions**: Déjà convertis

### Ad Sets

#### Ad Set D: Retargeting visiteurs
- **Message**: "Vous avez consulté Tennis Breakdown..."
- **Offer**: 20% première analyse
- **Urgency**: Offre limitée 7 jours

#### Ad Set E: Guide téléchargeurs
- **Audience**: Téléchargé guide gratuit
- **Sequence**: Email 3 → Ad complémentaire
- **Message**: "Prêt à passer à l'action? Voici l'étape suivante."

#### Ad Set F: Page viewers spécifiques
- **Audience**: Vu page tarifs mais pas converti
- **Message**: "Questions sur notre service? FAQ complète ici."
- **CTA**: Voir FAQ

### Ads Creative
#### Format 1: FAQ vidéo
- **Durée**: 60 secondes
- **Content**: Sami répond à question commune
- **Authenticity**: Visage caméra, expertise visible

#### Format 2: Prix & valeur
- **Image**: Comparaison visuelle prix/valeur
- **Copy**: "Coaching traditionnel €50-100 vs Notre analyse €25"
- **Social proof**: "50+ joueurs déjà améliorés"

#### Format 3: Garantie
- **Format**: Image texte simple
- **Message**: "Satisfait ou remboursé sous 7 jours"
- **Trust signal**: Icône garantie, badges sécurité

### Budget & Schedule
- **Budget**: €15/jour (€105/semaine)
- **Duration**: Continue
- **KPI**: ROAS > 2, CPA < €25
- **Scaling**: Augmenter budget si ROAS > 3

## Campagne 3: Performance (Conversion)

### Audience
- **Hot leads**: Email ouvert, guide téléchargé, site multiple visites
- **Lookalike 2%**: Expansion audience performante
- **Exclusions**: Déjà acheté

### Ad Sets

#### Ad Set G: Offre lancement
- **Audience**: Tous leads chauds
- **Offer**: €15 première analyse (vs €35 normal)
- **Limited**: "50 premières analyses à ce prix"

#### Ad Set H: Urgency
- **Audience**: Abandonné checkout
- **Message**: "Votre offre spéciale expire dans 24h"
- **Timer**: Countdown intégré

#### Ad Set I: Social proof massif
- **Audience**: Large mais qualifié
- **Content**: Compilation témoignages
- **Social proof**: "Rejoignez 50+ joueurs satisfaits"

### Ads Creative
#### Format 1: Checkout simplifié
- **Message**: "2 minutes pour améliorer votre tennis"
- **Visual**: Interface checkout simplifiée
- **CTA**: Commencer maintenant

#### Format 2: Témoignage long
- **Format**: Vidéo 90 secondes
- **Content**: Interview client (autorisation)
- **Storytelling**: Problème → Solution → Résultats

#### Format 3: Comparaison avant/après détaillée
- **Format**: Side-by-side vidéo
- **Analysis overlay**: Textes explicatifs
- **Results metrics**: "Service vitesse +15km/h"

### Budget & Schedule
- **Budget**: €20/jour (€140/semaine)
- **Objective**: Conversions
- **KPI**: CPA €18, ROAS 3+
- **Automation**: Dynamic ads checkout abandon

## Tests A/B Planifiés

### Test 1: Ad Copy
- **Variable**: Problème-focused vs Solution-focused
- **Hypothèse**: Solution convertit mieux à stage consideration
- **Sample size**: 500 impressions par variation

### Test 2: Visual
- **Variable**: Vidéo vs Carousel vs Image simple
- **Hypothèse**: Vidéo engagement plus haut mais carousel conversion mieux
- **Duration**: 5 jours

### Test 3: Offer
- **Variable**: 20% off vs €15 flat vs Bundle
- **Hypothèse**: €15 flat plus clair, meilleur conversion
- **Audience**: Split leads chauds

### Test 4: CTA
- **Variable**: "Télécharger guide" vs "Découvrir erreurs" vs "Améliorer maintenant"
- **Hypothèse**: Action-oriented performe mieux awareness
- **Stage**: Awareness campaign

## Budget Total Phase 1 (30 jours)

### Répartition
- **Awareness**: €210 (30%)
- **Consideration**: €315 (45%)
- **Conversion**: €420 (25%)
- **Total**: €945

### ROI Projection
```
Hypothèse:
- Leads: 500 (coût €1.89/lead)
- Conversions: 50 (taux 10%, CPA €18.90)
- Revenue: 50 × €20 = €1,000
- ROI: €1,000 / €945 = 1.06 (break-even)
```

### Scaling Criteria
- **CPA acceptable**: < €25
- **ROAS minimum**: 1.5
- **Volume minimum**: 3 conversions/jour
- **Scaling**: +20% budget/semaine si critères remplis

## Calendrier d'Exécution

### Jour 1-3: Setup
- Pixel installation vérification
- Audience création
- Ads création (3 formats × 3 campagnes)
- Budget allocation

### Jour 4-10: Test Phase
- Lancement 3 campagnes
- Monitoring quotidien
- A/B tests début
- Optimisation premières 48h

### Jour 11-20: Optimisation
- Pause ad sets non performants
- Scale winners
- Implement retargeting
- Launch new creatives

### Jour 21-30: Scaling
- Expand lookalike audiences
- Increase budget performants
- Test new angles
- Analyze full funnel

## Contingency Plan

### Si CPA trop élevé (>€30)
1. Pause campaign, analyse audience
2. Test new creatives moins "salesy"
3. Focus organique + SEO temporarily
4. Re-launch with refined targeting

### Si volume faible (<1 conversion/jour)
1. Widen audience parameters
2. Test different offer (lower price point)
3. Improve landing page conversion rate
4. Add more social proof elements

### Si ROAS bon mais volume limité
1. Scale budget gradually (+20% weekly)
2. Expand to Instagram placements
3. Test Google Ads complementary
4. Explore influencer partnerships
