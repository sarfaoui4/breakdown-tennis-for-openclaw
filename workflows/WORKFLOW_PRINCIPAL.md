# Workflow Client Acquisition → Livraison

## Étape 1: Découverte (Marketing)
```
[Public] → [Site/RS] → [Intérêt] → [Landing Page] → [Lead]
```
- **Agent Marketing**: SEO, ads, contenu
- **Agent Copy**: Landing page copy, emails capture
- **Outils**: Analytics, A/B testing

## Étape 2: Soumission (Réception)
```
[Lead] → [Formulaire Upload] → [Vidéo Soumise] → [Confirmation]
```
- **Agent Réception**: Accueil, vérification format
- **Email automatique**: Confirmation + délai estimé
- **Ticket créé**: Système de suivi

## Étape 3: Traitement (Production)
```
[Ticket] → [Préparation] → [Analyse Sami] → [Rédaction]
```
- **Agent Analyse**: Préparation vidéo, outils
- **Sami**: Analyse experte (coeur métier)
- **Agent Copy**: Assistance rédaction format standard

## Étape 4: Livraison (Service)
```
[Analyse Finalisée] → [Formatage] → [Email Livraison] → [Client]
```
- **Agent Réception**: Formatage livrable
- **Email**: Livraison + instructions
- **Système**: Marquer comme complété

## Étape 5: Suivi (Support)
```
[Client] → [Feedback/Questions] → [Support] → [Rétention]
```
- **Agent Support**: Réponses questions
- **Collecte feedback**: Amélioration service
- **Upsell**: Analyses suivantes

## Rôles & Responsabilités

### Sami (CEO/Expert)
- Analyse vidéo finale
- Validation qualité
- Décisions stratégiques
- Relations clients premium

### Claude (COO/Coordinateur)
- Supervision agents
- Optimisation workflows
- Reporting performance
- Coordination équipe

### Agents IA
- Exécution tâches spécialisées
- Reporting quotidien
- Escalade problèmes
- Amélioration continue

## KPI par Étape
1. **Acquisition**: Coût client, taux conversion
2. **Soumission**: Taux abandon, qualité vidéos
3. **Traitement**: Temps moyen, satisfaction Sami
4. **Livraison**: Temps livraison, qualité perçue
5. **Suivi**: Taux rétention, feedback score

## Automatisation Progressive
- Phase 1: Sami manuel, agents assistance
- Phase 2: 50% automatisé (préparation, templates)
- Phase 3: 80% automatisé (analyse suggérée, validation Sami)
- Phase 4: Équipe agents autonome, Sami supervision
